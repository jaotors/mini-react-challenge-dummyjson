# React Mini challenge G360
A simple React project built as part of a coding challenge.
This app demonstrates how to fetch, search, and filter data from the DummyJSON API.

### Clone the repository
git clone https://github.com/jaotors/mini-react-challenge-dummyjson.git

### Navigate into the project folder
cd mini-react-challenge-dummyjson

### Install dependencies
npm install

### Start the development server
npm run dev
